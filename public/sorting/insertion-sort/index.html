Insertion Sort

Insertion Sort Algorithm.

  • 0.5 x
  • 0.75 x
  • Normal
  • 1.25 x
  • 1.5x
  • Random
  • Reverse
  • Few unique
  • Nearly Sorted

Algorithm

do

swapped = false

for i = 1 to indexOfLastUnsortedElement-1

if leftElement > rightElement

swap(leftElement, rightElement)

swapped = true; ++swapCounter

while swapped

Time Complexity : O(n*n)

Auxilary Space : O(1)
